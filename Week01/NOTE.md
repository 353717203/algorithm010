学习笔记
====
## 学习内容
### 数组，链表，跳表（元素需有序）
 * 链表时间复杂度：
   * prepend O(1)
   * append O(1)
   * lookup O(n)
   * insert O(1)
   * delete O(1)
 * 数组时间复杂度：
  * prepend O(1)
  * append O(1)
  * lookup O(1)
  * insert O(n)
  * delete O(n)
* 栈，队列，优先队列，双端队列
* 时间复杂度和空间复杂度分析（完成算法代码之前需清除所写算法的时间复杂度和空间复杂度）
 * 常用时间复杂度表示：O(1)、O(log n)、O(n)、O(n^2)、O(n^3)、O(2^n)、O(n!)
 * 二叉树遍历、图的遍历时间复杂度：O(n) --遍历所有节点
 * 深搜(DFS)、广搜(BFS)时间复杂度：O(n)
 * 二分查找：O(log n)
 * 归并排序：O(n log n)
